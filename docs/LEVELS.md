# Система уровней (Levels 1-40)

## Обзор

Уровни теперь **хранятся в базе данных Supabase** для безопасности. Пользователи не имеют доступа к коду и не могут манипулировать порогами уровней.

## Прогрессия

- **Уровни 1-7**: Удваивается каждый раз (×2)
- **Уровни 8-10**: Переход к круглым числам (250, 500, 1000)
- **Уровни 11-40**: Повторяется паттерн удвоения с круглыми числами

## Безопасность

✅ **Уровни хранятся в Supabase** - пользователи не могут изменить пороги
✅ **RLS политики** - только чтение для всех
✅ **Fallback значения** - если БД недоступна, используются первые 10 уровней

## Миграция

Выполните миграцию:
```bash
# Примените миграцию в Supabase Dashboard или через CLI
supabase db push
```

Или вручную выполните SQL из файла:
`supabase/migrations/20250126_create_level_thresholds.sql`

## Использование

```typescript
const { levelThresholds, isLoading } = useLevelCheck()

// levelThresholds загружаются из БД автоматически
// isLoading показывает статус загрузки
```
